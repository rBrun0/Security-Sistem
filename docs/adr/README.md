# ADRs (Architecture Decision Records)

Este diretório guarda decisões arquiteturais relevantes do projeto.

## Como usar

- Use o template em [0000-template.md](0000-template.md).
- Nomeie novos arquivos com numeração incremental: `0001-...`, `0002-...`.
- Mantenha decisões curtas, objetivas e focadas em trade-offs.
- Ao mudar uma decisão, crie nova ADR (não sobrescreva a antiga).
